Puntatori in C

Introduzione ai puntatori

  • I puntatori sono una delle caratteristiche più potenti e complesse del linguaggio C.
  • I puntatori sono utilizzati per manipolare direttamente la memoria del computer.
  • Per comprendere i puntatori, dobbiamo prima capire come funziona la memoria del computer.

La memoria del computer

  • La memoria del computer è una sequenza di celle di memoria.
  • Ogni cella di memoria ha un indirizzo univoco che la identifica.
  • Ogni cella di memoria può contenere un valore.
  • Il tipo di dato determina quanti byte occupa una cella di memoria.

Dichiarazione di un puntatore

  • Un puntatore è una variabile che contiene l'indirizzo di una cella di memoria.
  • La dichiarazione di un puntatore richiede un tipo di dato.
  • Il tipo di dato del puntatore deve corrispondere al tipo di dato del valore che si vuole puntare.
  • La sintassi: 
              tipo_dato *nome_puntatore;

Operatore di indirizzo &

  • L'operatore di indirizzo & restituisce l'indirizzo di una variabile.
  • Per assegnare l'indirizzo di una variabile a un puntatore, utilizziamo l'operatore di indirizzo &.
  • Esempio: 
              int a = 10;
          int *p = &a;
          printf("Il valore di a è %d\n", a);
          printf("L'indirizzo di a è %p\n", &a);
          printf("Il valore puntato da p è %d\n", *p);

Operazioni con i puntatori

  • Un puntatore può essere incrementato o decrementato.
  • L'operazione di incremento o decremento dipende dal tipo di dato puntato dal puntatore.
  • Esempio: 
              int a[] = {1, 2, 3, 4, 5};
          int *p = a;
          printf("Il primo elemento di a è %d\n", *p);
          p++;
          printf("Il secondo elemento di a è %d\n", *p);

Puntatori a puntatori

  • È possibile dichiarare un puntatore a un puntatore.
  • Un puntatore a un puntatore contiene l'indirizzo di un puntatore.
  • Esempio: 
              int a = 10;
          int *p = &a;
          int **pp = &p;
          printf("Valore di a: %d\n", a);  // Output: Valore di a: 10
          printf("Valore puntato da p: %d\n", *p);  // Output: Valore puntato da p: 10
          printf("Valore puntato da pp: %d\n", **pp);  // Output: Valore puntato da pp: 10
